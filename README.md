# Rajarajan Ponraj - Portfolio Website

This is a premium personal portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Tech Stack
- **Next.js**: Framework
- **Tailwind CSS 4**: Styling
- **Framer Motion**: Animations
- **Lucide React**: Icons

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3. **Build for Production**
   ```bash
   npm run build
   ```

## Customization

Edit `lib/data.tsx` to update your personal information, skills, projects, and social links.
